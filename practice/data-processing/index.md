

# Le traitement de données dans les architectures BIG DATA 

## Overview

L'objectif est de manipuler les outils et technologies associées aux architectures Big Data: 
Les notions abordées sont les suivantes:

* Introduction au Big Data (Prérequis)
* Hadoop et Map Reduce
* Traitement de données (Batch, Streaming, Temps Réel, etc.)
* Bases de données NoSQL

Ce dossier comporte quatre travaux pratiques:

* TP1: Hadoop et Map Reduce
* TP2: Apache Spark
* TP3: Apache Kafka
* TP4: HBase
